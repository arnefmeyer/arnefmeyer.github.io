Bases: object

A data set information meta class

Contains meta data of a specific data set. Data can be read by passing a DatasetInfo object to the Database’s read method

Parameters:
  • experiment (string) – The name of the experiment
  • condition (string) – Experimental condition, e.g., stimulus
  • block (string) – The name of the experimental block, e.g., the name of the measurement (including data)
  • channel (int) – Electrode channel index
  • unit (int) – Index of spike-sorted unit
  • types (list) – A list containing the data to be loaded from the database, e.g., “Stimulus”, “SpikeUnit” etc.
  • string (string) – A string representing the data set as returned by the “to_string()” method

lnpy.util.resample(x, fs_old, fs_new, axis=0, algorithm='scipy')¶

Resample signal

If available resampling is done using scikit samplerate. Otherwise, scipy’s FFT-based resample function will be used.

Converters available in scikits.samplerate: - sinc_medium - linear - sinc_fastest - zero_order_hold - sinc_best
